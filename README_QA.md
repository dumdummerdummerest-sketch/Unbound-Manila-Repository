Installation of Playwright:
- npm install -D @playwright/testnpm install -D @playwright/test
- npx playwright install


Running files:
- Open the server in another tab of the command prompt, in the same folder directory
- then test in another tab using: npx playwright test <path> 
- Example: npx playwright test tests/auth/login.spec.js
- Note: Make sure you are in the main project directory
